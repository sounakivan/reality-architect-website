Visual Poetry Generator

Three.js WebGL Generative Art Text Analysis

A generative art system that transforms text into living visual forms. The system analyzes emotional content, word patterns, and semantic meaning to create unique 3D visualizations that pulse, flow, and evolve.

üìù What I Explored: How can abstract concepts like emotion, rhythm, and meaning be translated into visual form? This experiment explores the boundary between language and visual representation.

üéØ What I Learned: Creating meaningful connections between text and visuals requires both technical skill and artistic intuition. I learned to map linguistic features (sentiment, complexity, emotional intensity) to visual parameters (color, motion, form complexity).

How It Works

The visualization engine analyzes your text for several key features:

  • Emotional Valence: Positive/negative sentiment affects color temperature and brightness
  • Complexity: Sentence structure and vocabulary influence form complexity
  • Rhythm: Word patterns create visual rhythm through motion and repetition
  • Intensity: Emotional intensity drives particle density and movement speed

Technical Implementation

In a full production version with AI API integration:

  • NLP Analysis: GPT-4 or Claude analyzes text for semantic meaning, emotional content, and linguistic patterns
  • Parameter Mapping: AI outputs are mapped to visual parameters (particle count, color palettes, motion patterns)
  • Real-time Generation: Three.js renders dynamic 3D visuals that evolve based on the analysis
  • Interactive Response: Users can interact with the visualization, affecting its evolution

This demo uses client-side text analysis to create responsive visualizations without requiring API calls, demonstrating the core concept.
