Automated YouTube Content Pipeline

n8n ElevenLabs Airtable GPT-4 Google Drive Trello

Overview

This system (built in n8n) automates the journey from "video idea" to "ready-to-edit assets" using AI agents and project management tools, with a critical human checkpoint.

Workflow Steps

1. Initiation & Ideation

Trigger: The workflow starts on a set schedule.

Action: It searches an Airtable Database to retrieve a specific video concept/idea and supporting details.

2. AI Content Generation

Scriptwriting: An AI Agent (powered by GPT-4.1) takes the idea and generates a full video script.

Metadata: The agent simultaneously writes the video description and formats the output.

3. Organization & Documentation

Google Drive: The system automatically creates a new Google Doc, pastes the generated script, and moves it to a designated "Script Folder."

Project Management: It creates a new Trello card for the project and attaches the script file for easy access.

4. Human-in-the-Loop

The Pause: The automation halts for a manual review.

Action: A human receives a message to proofread the script. The workflow does not proceed until this step is approved.

5. Audio Production & Finalization

Voiceover Generation: Once approved, the system pulls the script and sends it to ElevenLabs api for voice generation.

Asset Consolidation: The audio file is uploaded to Google Drive and attached to the existing Trello card.

6. Completion

The system updates the original Database record to mark the phase as complete. This helps the project manager know to send the materials for this video idea to the video editor.
